# Ejercicio-Python-
Hello, today I’m going to explain a small Python project that I created.
The purpose of this project is to register product sales and calculate a simple sales summary.

The program allows a user to enter information about products that were sold. For each sale, the user types the product name, the unit price, and the quantity that was sold. The program can store multiple sales, so the user can keep adding products until they decide to stop.

The project is divided into three different files to keep the code organized. Each file has a different responsibility.

The first file is main.py. This is the main part of the program. It interacts with the user by asking for the product name, the price, and the quantity. It also stores all the sales information in a list.

The second file is calcular.py. This file is responsible for the calculations. It reads the sales data and calculates two things: the total quantity sold for each product and the total revenue generated from all sales.

The third file is mensaje.py. This part of the program displays the final results. It prints a summary showing each product and how many units were sold, and it also shows the total revenue from all the sales.

In summary, this project demonstrates how a simple sales system can be created using Python. It collects sales data, processes the information, and then displays a clear summary for the user.
